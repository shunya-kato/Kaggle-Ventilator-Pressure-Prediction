# Kaggle-Ventilator-Pressure-Prediction

## Settings
```
pip install -r requirements.txt
```
Create a data folder and a logs folder as follows:
```
│
├ configs
├ data 
    ├ input 
        ├ train.csv
        ├ test.csv
    ├ output
        ├ sample_submission.csv
├ logs
    ├ fold0
    ├ fold1
    ├ fold2
    ├ fold3
    ├ fold4
```
